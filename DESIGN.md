# StudioFlow — Design Guidelines

## Brand Identity
- App Name: StudioFlow
- Logo: Bold SF monogram with dynamic motion arc in electric blue and emerald green
- Primary Colors: Electric blue (#2563EB) and emerald green (#10B981)
- Accent: Vibrant gradient from blue to green for CTAs and highlights

## Typography
- Headlines: Bold, modern sans-serif (Inter)
- Body: Clean, readable at small sizes
- Tone: Professional, energetic, confident

## Visual Direction
- Clean, minimal admin dashboard aesthetic
- Card-based layouts for class schedules and member summaries
- Data-dense but uncluttered — owners need to scan quickly
- Mobile-responsive (owners check on the go)
- Status indicators: green (full/active), amber (filling/at risk), red (empty/churning)

## Component Style
- shadcn/ui primitives throughout
- Rounded corners (rounded-xl), subtle shadows (shadow-sm)
- Sidebar navigation for main sections: Dashboard, Classes, Members, Trainers
- Tables for member lists; calendar/grid view for class schedules
- Charts/sparklines for retention and fill-rate trends on the dashboard
