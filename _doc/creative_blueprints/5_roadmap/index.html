What's Next

Three Features Launching After MVP

Built on the class scheduling foundation — each goal deepens the core mission: fill every class and keep every member.

Goal 1

Member Retention Alerts

Automated early-warning system that catches disengaging members — missed classes, expiring memberships, falling booking frequency — and surfaces them before they churn.

  • Configurable alert thresholds per signal type
  • At-risk dashboard sorted by urgency
  • One-tap re-engagement actions from the alert view
  • Alert history and resolution tracking per member
Outcome: catch churn before it's a cancellation
Goal 2

Trainer Management Hub

Unified trainer operations: profiles, availability calendars, class assignment tracking, and conflict detection — so owners always know who's available, who's overloaded, and who's covering what.

  • Trainer profiles with certifications and specialties
  • Editable availability calendar per trainer
  • Class assignment tracking across the full schedule
  • Automatic conflict and overload detection
Outcome: zero scheduling conflicts, full coverage
Goal 3

Self-Service Member Booking Portal

A member-facing view: browse the schedule, book or cancel spots, join the waitlist, and check membership status — reducing front-desk load and increasing fill rates through frictionless self-service.

  • Browsable schedule with live seat availability
  • One-tap book, cancel, and waitlist join
  • Membership status and booking history view
  • Automatic spot promotion notifications
Outcome: higher fill rates, less front-desk admin
