# StudioFlow — Product Overview

## What It Is
StudioFlow is the all-in-one management platform for fitness studio owners. Schedule classes, track memberships, coordinate trainers, and fill every seat — all from one dashboard built to boost retention and revenue.

## Target Users
Fitness studio owners and managers running boutique gyms, yoga studios, pilates centers, or CrossFit boxes. Typically managing 5–30 trainers and 50–500 active members who need to fill classes and reduce churn.

## Core Problem
Studio owners have no unified tool to manage class schedules, track member attendance and retention, or coordinate trainer availability. Empty class slots go unfilled, members churn silently, and trainers get double-booked — costing the studio real revenue every week.

## Day-One Feature
Class scheduling with live seat availability — studio owners can create and manage class slots, assign trainers, and see real-time booking counts so they can act on empty seats before a class runs.

## Strategic Goals
- Boost member retention by surfacing at-risk members early
- Fill every class slot with visibility into booking rates
- Simplify trainer coordination and prevent scheduling conflicts
- Give studio owners a single command center for their operations
